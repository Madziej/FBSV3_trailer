About the Project

The primary goal behind creating Fantasy Battle Simulator is educational — driven by a desire to gain programming skills. The current version is developed using widely accessible tools, such as Visual Studio Code, and languages like JavaScript. Naturally, HTML and CSS are included as foundational technologies essential for creating a user-friendly interface that enhances the player’s experience.

Previous Versions

1. Initial Java Drafts (January 2024) The earliest concepts for the engine were created in Java, featuring simple console applications without any graphical interface. The basic version of the Simulator could only run within a compiler program, such as IntelliJ. A few recordings document this initial phase and can be viewed below. Be careful. They were written in polish laguage.

2. Fantasy Battle Simulator V1 (May 2024)

The next milestone was presenting the concept in a form that could be shared with a wider audience. Backend languages alone couldn’t achieve this, prompting me to learn frontend development and become proficient in JavaScript. Version 1 introduced a simple, randomized battle system. The program generates statistics for two random fighters and independently simulates the battle. The player doesn’t control the process; it’s more like observing an illegal underground animal fight. Users can only follow a real-time text log that reports the battle’s progress. The design was minimalistic, but I was thrilled to have created a web application that met my basic goals.

3. Fantasy Battle Simulator V2 (July 2024)

In Version 2, players gained the ability to create their own character and select an enemy from a list sorted by statistics. This version marked a significant step toward a more complex engine with enhanced visual elements.

In particular:

- Fighters now have dynamic dialogues that change based on the situation on the battlefield.

- Health bars respond in real-time, reflecting each hit taken.

- Characters feature unique avatars, customized by race and class.

Current Drawbacks:

The page is not fully responsive. To run the app on a phone, you’ll need to rotate it 90° for optimal viewing. Some elements aren’t properly scaled, requiring you to scroll during battles to track features like the health bar.

Fantasy Battle Simulator V2

4. Fantasy Battle Simulator V3 – Sci-Fi Edition (in progress)

This version marks a significant evolution of the Fantasy Battle Simulator in several ways:

- Active Player Involvement: The player will (finally) move beyond passive observation, gaining control over their character’s strategy and actions during combat.

- Pause Functionality: A new "pause" button will allow players to pause and strategize mid-battle.

- Expanded Battles: Combat will include more than just two fighters. The player, alongside allied NPCs, will face off against groups of hostile enemies, creating a more dynamic battlefield experience.

- Campaign Mode: Unlike previous versions, V3 will offer a brief campaign with three unique missions, each featuring distinct enemy types and objectives.

- Character Progression: After each mission, the player's character will level up, allowing for incremental customization and skill development throughout the space adventure.

- Enhanced Character Creation: Character creation will offer a wider array of options than in V2, allowing for more personalized and diverse characters.
